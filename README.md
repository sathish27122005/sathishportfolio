# Sathish - Future Tech Portfolio

A modern, futuristic portfolio website built with vanilla HTML, CSS, and JavaScript. This project showcases advanced UI interactions, 3D elements, and creative physics effects.

## üöÄ Features

*   **3D Image Gallery**: An interactive, rotating 3D carousel for showcasing projects.
*   **Gravity Effect**: A "Do Not Touch" button that unleashes a physics simulation, causing page elements to fall and collide.
*   **Glassmorphism UI**: Sleek, modern, semi-transparent design elements.
*   **Project Details**: Dynamic routing to detailed project pages with feature lists and technology tags.
*   **Certificate Modal**: Interactive modals to display certifications and key learnings.
*   **Feedback System**: Integrated contact form with a 5-star rating system that links directly to Gmail.
*   **Responsive Design**: Optimized for desktop and mobile viewing.
